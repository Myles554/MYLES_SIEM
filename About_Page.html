About MYLES_SIEM

MYLES_SIEM is a personal cybersecurity project created to understand how modern security monitoring platforms operate.

The project is not intended to replace enterprise SIEM solutions. Instead, it serves as a practical learning environment for studying log management, event correlation, alerting systems, dashboards, and security operations workflows.

MYLES_SIEM

This is a student built Security Monitoring Platform

Welcome to MYLES_SIEM, a cyber security learning project designed to explore how Security Information and Event Management Systems(SIEM) work. The Goal of MYLES_SIEM is to understand technologies and workflows used by modern security operations centers.

The project combines log analysis, security monitoring, alert management, and incident response concepts while using Wazuh as the underlying SIEM engine. This project is being developed step by step as a learning journey in cybersecurity engineering, Linux administration, Python development, networking, and security operations.

Project Goals

  1. Learn SIEM architecture and design.
  2. Understand log collection and monitoring.
  3. Explore threat detection workflows.
  4. Develop practical cybersecurity engineering skills.
  5. Build a portfolio project for future cybersecurity roles.
